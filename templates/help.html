📚 MP4 Creator 도움말

🎯 앱 소개

MP4 Creator는 텍스트를 자동으로 음성과 이미지가 포함된 MP4 동영상으로 변환해주는 도구입니다.

🚀 주요 기능

🎤 음성 변환 (TTS)

텍스트를 자연스러운 한국어 음성으로 변환

  • 3가지 음성 선택 (남자1, 남자2, 여자1)
  • 세밀한 속도 조절 (0.01 단위)
  • 특수 명령어 지원

📊 구글 스프레드시트 연동

대본을 스프레드시트에서 자동으로 읽어옴

  • 대본 목록 자동 읽기
  • 이미지 정보 관리
  • 실시간 동기화

🖼️ 이미지 검색

Giphy API를 통한 자동 이미지 검색

  • 키워드 기반 GIF 검색
  • 고품질 이미지 선택
  • 자동 다운로드

🎬 MP4 생성

음성과 이미지를 결합한 동영상 제작

  • 자막 타임라인 생성
  • 고품질 동영상 출력
  • 구글 드라이브 자동 업로드

📝 사용 방법

1

설정 구성

설정 페이지에서 Google API 키, Giphy API 키, 음성 설정 등을 구성합니다.

2

대본 작성

Google 스프레드시트에 대본을 작성하거나 직접 텍스트를 입력합니다.

3

MP4 생성

앱에서 생성 버튼을 클릭하면 자동으로 음성, 이미지, 동영상이 생성됩니다.

🔧 특수 명령어

[목소리:남자1] 음성을 남자1로 변경 (남자1, 남자2, 여자1)
[속도:+25%] 말하기 속도를 25% 빠르게 설정
[속도:-25%] 말하기 속도를 25% 느리게 설정
[쉼:2.0] 2초 쉼 추가 (그 자리에서 즉시 적용)
[검색어:고양이] 고양이 이미지 검색

📝 명령어 사용 규칙

목소리/속도 명령어: 줄바꿈 전까지 계속 적용됨
쉼 명령어: 해당 위치에서 즉시 무음 삽입
속도 범위: -50% (절반속도) ~ +100% (2배속), 부호 있는 백분율로 입력
예시: [목소리:남자1][속도:-20%]안녕하세요 [쉼:1.0] 반갑습니다

⚙️ 필수 설정 항목

Google 설정

  • 서비스 계정 키 파일 경로
  • 이미지 디렉토리 ID
  • 스프레드시트 ID들

Giphy 설정

  • Giphy API 키
  • 검색 결과 제한 수

음성 설정

  • 기본 음성 선택
  • 기본 속도 설정

❓ 자주 묻는 질문

Google API 키는 어떻게 얻나요?

Google Cloud Console에서 새 프로젝트를 만들고, Google Sheets API와 Google Drive API를 활성화한 후 서비스 계정을 생성하여 JSON 키 파일을 다운로드하세요.

Giphy API 키는 어디서 받나요?

Giphy Developers 사이트(developers.giphy.com)에서 무료 계정을 만들고 API 키를 발급받을 수 있습니다.

생성된 MP4 파일은 어디에 저장되나요?

설정에서 지정한 Google 드라이브 폴더에 자동으로 업로드됩니다.

음성 품질을 높이려면?

속도를 적절히 조절하고, 문장 사이에 쉼표나 마침표를 적절히 사용하세요. 특수 명령어로 세밀한 조정도 가능합니다.
